# 🗂️ Workspace

Un proyecto = una carpeta `workspace/<slug>/` con su nota principal `<slug>.md` (schema `project`) y notas satélite libres. Plantilla: `_templates/project.md`.

Ciclo de vida: `idea → active → paused / done → archived`.
