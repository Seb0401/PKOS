# TraceReq — Gestión y trazabilidad de requerimientos

## Resumen

Plataforma web para gestionar requerimientos de software: funcionales (RF) y no funcionales (RNF) con identificadores autogenerados, prioridades, historial de cambios, casos de uso vinculados (CU) y trazabilidad entre requerimientos (dependencia, refinamiento o contradicción). Incluye matriz de trazabilidad con grafo de dependencias, dashboard de métricas y exportación CSV/JSON.

Backend en [[flask]] con SQLAlchemy sobre [[mysql]], organizado en blueprints; vistas con Jinja2.

## Objetivos

- Cubrir el ciclo completo: requerimiento → caso de uso → trazabilidad → métricas.
- Multiproyecto, con historial de cambios auditable.
