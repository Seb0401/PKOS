# Codventure — RPG web para dominar estructuras de datos y algoritmos

## Resumen

RPG educativo en el navegador para aprender programación competitiva: 10 actos y ~40 módulos desde arrays hasta flujo máximo y geometría computacional, con tres niveles por módulo (Bronze/Silver/Gold). Filosofía **"Forge before Shop"**: hay que implementar cada estructura desde cero antes de desbloquear la versión de librería.

Juez de código *en el navegador* (sin servidor): Web Workers para JavaScript, Pyodide para Python y clang-wasm (experimental) para C++. Progresión con XP, rangos de gremio y repetición espaciada.

## Objetivos

- MVP Fase 1: primer acto jugable con juez en navegador.
- Que el propio desarrollo sirva para practicar lo que el juego enseña (DSA en TypeScript).
